# 03. TERMINAL_WORKFLOW

Dette dokument beskriver Yggdras automatiserede terminal-setup i VS Code. Formålet er at fjerne manuel opsætning ved hver sessionsstart.

## 1. Konfiguration
- **Fil:** `.vscode/tasks.json`
- **Trigger:** `runOn: folderOpen`
- **Værktøj:** VS Code Tasks API

## 2. Automatiske Terminaler
Ved åbning af Yggdra-workspace i VS Code oprettes følgende terminaler automatisk:

1. **Bash (Lokal):** Standard shell i roden af repoet.
2. **VPS SSH:** Direkte forbindelse til driftsmiljøet (srv1181537).
3. **Qdrant Tunnel:** SSH-tunnel (port 6333) til den eksterne Vector Database.

## 3. Vedligeholdelse
- Hvis IP-adresser eller SSH-keys ændres, skal `.vscode/tasks.json` opdateres.
- Terminalerne kører i baggrunden (`isBackground: true`) for at minimere visuel støj ved start.
