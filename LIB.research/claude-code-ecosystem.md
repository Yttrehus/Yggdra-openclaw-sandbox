# Research: Claude Code Ecosystem & Agentic Patterns

## 1. Executive Summary
Claude Code representerer et skift fra "chat-om-kode" til "agent-i-terminalen". For Yggdra betyder det et muligt skift i, hvordan Lag 3 (Handling) eksekveres lokalt.

## 2. Key Features
- **Context Awareness:** Bedre forståelse af lokale projektstrukturer end standard LLM API'er.
- **Tool Execution:** Evne til at køre tests, rette fejl og committe ændringer direkte.
- **Harness Patterns:** Brugen af MCP (Model Context Protocol) til at forbinde Claude til eksterne data (Notion, Google Calendar).

## 3. Strategisk Betydning for Yggdra
Integration af Claude Code patterns kan accelerere vores V6 mål om reelle API-kald ved at bruge eksisterende MCP servere til Google Workspace.
