# Prompt-skabeloner

Genanvendelige instruksskabeloner for workflows Yttre bruger gentagne gange. Ikke et katalog — genveje.

@CONTEXT.md

## Kontekst

- Kilde: Yttres chatlog (PC + VPS), eksisterende skills som format-reference
- Stack: Claude Code skills (SKILL.md + references/), markdown
- IKKE i scope: eksterne prompt-biblioteker, MCP-research (det er mcp-skills-kompendium), VPS-data-organisering (det er ydrasil)

## Kommunikation

- Yttre gør selv, Claude guider fra sidelinjen
- Kort og direkte — gentag til done
- State på disk, aldrig kun i chatten

## Workflow

- CONTEXT.md opdateres ved pauser, beslutninger, session-slut
- Commit + push efter hver logisk ændring
- Spørg før du bygger. Diskussion færdig → bekræftelse → kode.
