# Manuals

Levende håndbøger for daglig brug. Startede som cheatsheets, vokser til operationsmanualer for Yggdra.

## Indhold

| Fil | Emne | Oprindelse |
|-----|------|-----------|
| `git.md` | Git kommandoer, workflow, ordliste | M1 |
| `vscode.md` | VS Code genveje, settings, extensions | M2 |
| `terminal.md` | Shell aliases, installeret software, config-filer | M3 |
| `git-concepts.md` | Git-koncepter og mentale modeller | M1 |

## Regler

- Disse filer er **levende** — opdatér dem når noget ændrer sig.
- Hold dem korte og opslagbare. Formatér som tabeller og kodeblokke.
- Nye manualer tilføjes her når et nyt domæne er forstået nok til at dokumentere.
